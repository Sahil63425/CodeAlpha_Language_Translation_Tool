# CodeAlpha Internship | Task 1: Language Translation Tool
A lightweight, interactive web application that translates text across multiple languages and generates real-time audio output. Developed as part of the CodeAlpha Internship program.

Key Features
User Interface: Interactive web UI built using Gradio directly inside Google Colab.
Multilangual: Supports translation for English, Hindi, Urdu, Bengali, Spanish, French, and German.
Text-to-Speech (Audio): Automatically converts the translated text into spoken audio (.mp3) using gTTS.
Easy Copying: Includes a built-in one-click copy button for quick clipboard access.
Tech Stack
Language: Python 3
Libraries: Gradio, Deep-Translator, gTTS
Platform: Google Colab
How to Run
Open the .ipynb file in Google Colab.
Run the code cell (it will automatically install all required libraries).
